# Nuichat
This is an Open WebUI alternative that runs on node.js and npm
# How to run?
1. Download the Repo files as zip or git clone
2. Extract it
3. Use npm ci to install deps
4. Run with npm start and go to localhost:3000
# Availiable Providers
There are 17 availiable providers, they are OpenRouter, OpenAI, Anthropic, Google Gemini, Groq, DeepSeek, Mistral, Together AI, Fireworks AI, xAI, Cerebras, Perplexity, NVIDIA NIM, Cohere, Hugging Face, Deep Infra, Ollama.
But you can also use your own OpenAI compatible custom provider API key.
# Live (usable) website
Go to https://nuichat.onrender.com to use it without installing
